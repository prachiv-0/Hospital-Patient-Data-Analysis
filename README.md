Hospital Patient Data Analysis

Project Overview
This project focuses on analyzing hospital patient data to extract meaningful insights related to disease patterns, doctor workload, and treatment costs. The aim is to transform raw data into useful information that can support better decision making in healthcare management.

 Tools & Technologies Used
- Python (Pandas) – Data loading and analysis  
- Microsoft Excel – Pivot tables and data visualization  
- CSV Dataset – Source of patient data  

Dataset Description
The dataset contains the following fields:
- Patient ID  
- Age  
- Gender  
- Disease  
- Doctor  
- Treatment Cost   

Analysis Performed

 1. Disease Frequency Analysis
- Counted the number of patients for each disease  
- Helped identify the most common diseases  

 2. Treatment Cost Analysis
- Calculated statistical measures such as average, minimum, and maximum cost  

 3. Doctor Workload Analysis
- Grouped patients by doctor  
- Determined how many patients each doctor handled  

Data Visualization (Excel)

Pie Chart – Disease Distribution
- Created using Excel pivot table  
- Shows proportion of each disease among patients  

Bar Chart – Doctor Workload
- Displays number of patients treated by each doctor  
- Helps compare workload distribution  

Project Files
- `data.csv` – Dataset used for analysis  
- `data_analysis.py` – Python code for data analysis  
- `data.xlsx` – Excel file with pivot tables and charts  

Outcome
This project demonstrates how basic data analysis techniques can be applied to healthcare data to identify trends and support decision-making. It highlights the use of both Excel and Python for efficient data processing and visualization.

This project was developed as part of the **ExcelR Data Analytics Course Final Project**.
